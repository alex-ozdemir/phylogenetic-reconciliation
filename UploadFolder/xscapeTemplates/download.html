Download Xscape

The Xscape tools require Python 2.7. In addition, the tools use the Python packages listed below. These packages are standard in the Enthought Python distribution, which is free for academic users. The tools canalso be installed individually.

Packages Used:

  • matplotlib
  • shapely
  • BioPython
